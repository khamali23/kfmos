Web Application Development Course Outline

Web Application Development

Course Code: ITM 1131

Course Level: Year One, Semester One

Credit Unit: 4

Course Description

This course introduces students to Web technologies - HTML, CSS, JavaScript, CSS rameworks; JavaScript frameworks. Students will learn to implement modern web pages using HTML and CSS frameworks. The course also introduces students to full stack web application programming using Javascript.

Course Objectives

  1. Describe web technologies (HTML, CSS & JavaScript), CSS frameworks, and JavaScrpt frameworks.
  2. Implent a simple but modern web with HTML and CSS.
  3. Build a simple but functional web application using JavaScript.
  • Use web technologies (HTML, CSS & JavaScript), CSS frameworks, and JavaScript frameworks.
  • Design and create a simple but functional Web page using and CSS frameworks
  • Compose and build a simple full stack Web application using JavaScript framework.
